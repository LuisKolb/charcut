CharCut Web App
(go back)
Character-based MT evaluation and difference highlighting
This web app provides an interface to easily access and work with CharCut. To start:
    1. Paste your text passages into the two provided fields
    2. Match up sentences ("lines") you want to compare against each other
    3. Hit compare!
    4. If required, simply save the HTML Webpage to download the output file (CTRL+S)
The original CharCut paper can be found here:
Adrien Lardilleux and Yves Lepage: "CharCut: Human-Targeted Character-Based MT Evaluation with Loose Differences". In Proceedings of IWSLT 2017.
