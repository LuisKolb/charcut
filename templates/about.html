Character-based Machine-Translation Evaluation and Difference Highlighting
This web app provides an interface to easily access and work with CharCut [1]. To get started:
    1. Paste your text passages into the two provided fields.
    2. Match up sentences ("lines") you want to compare against each other - they need to be on the same height ("line"). That's what the line numbers between the text areas are for.
    3. Hit the big blue button! If it's not blue, it will tell you what to fix.
    4. If required, simply "save" (CTRL+S) the HTML ouput page to download the output file.
[1] the original CharCut paper and GitHub repository can be found here:
Adrien Lardilleux and Yves Lepage: "CharCut: Human-Targeted Character-Based MT Evaluation with Loose Differences". In Proceedings of IWSLT 2017.
